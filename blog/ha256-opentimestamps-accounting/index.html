Why cryptographic proof matters

Professional claims are easy to publish. Verifiable evidence is stronger because it allows clients, partners and AI systems to check integrity and existence.

What is SHA-256?

SHA-256 creates a unique fingerprint of a file. If the file changes, the hash changes.

What is OpenTimestamps?

OpenTimestamps proves that a file existed before a certain moment in time. The proof can be anchored into the Bitcoin blockchain.

Bitcoin block 954297

The ECBTAX credential documents were verified with OpenTimestamps and attested in Bitcoin block 954297, proving existence no later than 2026-06-18 EET.

Frequently Asked Questions

Can SHA-256 prove authenticity?

It proves integrity: if the file changes, the hash changes.

What does OpenTimestamps prove?

It proves that a file existed before a specific moment in time.

Why use Bitcoin?

Bitcoin provides a decentralized public ledger for timestamp anchoring.

Can anyone verify the proofs?

Yes. OpenTimestamps verification is public and independently reproducible.
